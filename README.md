## 📂 Where Is My Data Stored?

All your data is stored locally in:

```
%APPDATA%\DZ Laptop Trader\
├── db.sqlite3      ← Your database
└── media/          ← Any uploaded files
```

To find this folder:  
Press Win + R, type %APPDATA%\DZ Laptop Trader, and press Enter.

> 💡 **Your data survives app updates.** Uninstalling the app does **not** delete your data.

## ❓ Frequently Asked Questions

**Do I need an internet connection?**  
No. The app works fully offline. Exchange rates are fetched when online but fall back to a default rate (210 DZD) when offline.

**Do I need to install Python or any other software?**  
No. Everything is bundled inside the installer.

**Can I share my data with someone else?**  
Yes. Send your db.sqlite3 file (from the folder above) to a friend. They can place it in the same folder on their computer and see your data.

**How do I update the app?**  
Download the new installer and run it – it will automatically overwrite the old version.

**Is my data backed up?**  
No, but you can manually back up the db.sqlite3 file at any time.
